# Timeline

An interactive vertical timeline for logging and browsing dated events.

## Features

- Add events with a date, title, description, category, and color
- Events are sorted chronologically and displayed in an alternating left/right layout
- Scroll-reveal animation as events enter the viewport
- Edit or delete any event via hover actions on each card
- Six categories: Milestone, Meeting, Personal, Work, Learning, Other
- Eight color choices per event
- Data persists in **localStorage** — survives page reloads
- Pre-loaded with sample events on first launch
- Keyboard shortcut: `Ctrl/Cmd + N` focuses the title field; `Esc` closes the edit modal

## Usage

Open `Timeline.html` in any modern browser — no server needed.

## Notes

- Pure client-side HTML/CSS/JS
- All data stays in the browser's localStorage; nothing is sent externally
- Requires an internet connection only to load the Inter font from Google Fonts
